# CBase - 基础设施碳排放因子数据库

课题组共享的碳排放因子管理平台，支持多人协作录入、查询与管理排放因子。

## 功能

- **因子库** — 按类型浏览和搜索排放因子（能源燃料、物质材料、运输、机械设备、工艺过程、废弃回收）
- **因子详情** — 查看因子边界说明、时空适用范围、应用示例、注意事项
- **因子包** — 将因子组织到工程项目中，管理使用项，导出 CSV
- **数据库搜索** — 全局搜索因子

## 技术栈

- 前端：Vite + React
- 后端：Node.js + Express
- 数据库：SQLite

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
